# English Daily - 構想メモ

## コンセプト

毎朝6時に自動でその日のコンテンツを生成し、メールで届ける英語リスニング教材。
「英語を勉強する」ではなく「英語で新しい知識を得る」ことを目的とする。
まず自分1人が使えるものを作り、良ければ他の人にも展開できる設計にする。

---

## 体験の流れ

1. 毎朝6時、自動でニュース・天気などを取得
2. 指定した語彙レベルに合わせてAIが会話スクリプトを生成
3. メールで通知（成功ならリンク、失敗なら「今日は生成できませんでした」）
4. リンクを開くとウェブアプリで音声再生・スクリプト確認ができる
5. 過去のコンテンツは日付ごとに保存・後から聴ける
6. 再生位置は自動保存。次回「続きから再生しますか？」と表示

---

## 番組フォーマット

2人のホストによる朝のラジオ番組風の会話形式。

### キャラクター

**Alex**（男性・好奇心旺盛）
- テンションが高めで話題への反応が素直
- レベル外の単語が出たら「What does that mean?」と自然に聞く
- リスナーの代わりに質問してくれる役割

**Sarah**（女性・しっかり者）
- 知識が豊富で落ち着いている
- Alexの質問に会話の中でわかりやすく説明する
- たまにAlexのリアクションに笑いながら返す

## コンテンツカテゴリ

| カテゴリ | デフォルト | ソース |
|---------|----------|-------|
| 天気予報 | ON | Open-Meteo |
| 日本のニュース（英語） | ON | NHK World RSS（カテゴリ横断で幅広く） |
| 今日の出来事 | ON | Wikipedia「On This Day」 |
| 子育て・子どもの発達 | ON | HealthyChildren.org / Zero to Three（アーカイブからランダム） |
| 世界ニュース | OFF | Guardian API |
| スポーツ | OFF | 未定 |

重複対策：使用済み記事URLを記録し、同じ記事を連続で出さない。

## 技術構成（無料縛り）

| 役割 | 使うもの |
|------|---------|
| 天気 | Open-Meteo |
| ニュース | NHK World RSS / Guardian API |
| 歴史・雑学 | Wikipedia API |
| 子育て | HealthyChildren.org / Zero to Three RSS＋アーカイブ |
| スクリプト生成＋日本語訳 | Google Gemini API（無料枠1,500回/日） |
| 音声 | ブラウザ内蔵 Web Speech API（後で差し替え可） |
| メール送信 | Gmail SMTP |
| スケジューラー | GitHub Actions（UTC 21:00 ＝ JST 6:00） |
| ホスティング | GitHub Pages |
| コンテンツ保存 | GitHubリポジトリ（日付ごとJSON） |

---

## 拡張性の設計方針

- ユーザー情報・設定はすべて変数として外出し（コードに直接書かない）
- 人が増えたときは設定追加で対応できる構造
- 今は作らない：ログイン・認証・ユーザー管理・データベース

---

## 表示・UI

- **日本語字幕**：ボタン一つでON/OFF。Geminiがスクリプト生成時に日本語訳も同時に生成
- **テキスト選択可能**：スクリプトの文字は選択できる状態にしておく。iPhoneの内蔵辞書（「調べる」）がそのまま使えるので、アプリ内に辞書機能は不要
- **再生速度調整**：0.5倍〜2倍
- **再生位置の記憶**：続きから再生できる

---

## 将来の改善候補
